# AI CV Generator

An AI-powered job application toolkit that automatically generates tailored CVs and cover letters based on job descriptions.

## Features

- AI-powered CV tailoring using OpenAI
- Automatic bilingual CV generation (English + Hebrew)
- Professional PDF CV and Cover Letter export
- ATS keyword analysis
- Versioned CV generation per job
- Job-specific customization notes
- Streamlit UI for interactive CV creation

## Tech Stack

- Python
- Streamlit
- OpenAI API
- ReportLab (PDF generation)
- SQLite
- PowerShell automation

## What the system does

1. User pastes their base CV
2. User selects or pastes a job description
3. AI analyzes the job and tailors the CV
4. System generates:
   - English CV
   - Hebrew CV
   - Cover Letter
5. Files are exported as styled PDFs

## Why this project exists

Job applications require customizing CVs for each role.  
This tool automates that process while keeping the information truthful and structured.

It acts as a workflow system for managing job applications rather than just generating text.
